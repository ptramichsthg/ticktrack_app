   Digunakan untuk memantau seluruh sistem, melihat statistik menyeluruh, dan mengelola tiket.

2. ROLE: USER (PENGGUNA BIASA)

==================================================
CATATAN PENTING:
- Pastikan akun-akun di atas sudah Anda daftarkan ke dalam database MySQL Anda (tabel `users`).
- Karena aplikasi ini menggunakan `jBcrypt` untuk keamanan, password TIDAK BISA dimasukkan ke database dengan teks biasa (plain text).
- CARA TERMUDAH MENGAKALINYA: 
  1. Jalankan aplikasi TickTrack, lalu buat 2 akun baru di halaman *Register* menggunakan email di atas.
  2. Secara otomatis, kedua akun tersebut akan memiliki role 'user'.
  3. Buka database MySQL Anda (misal lewat phpMyAdmin).
  4. Ubah kolom `role` pada akun admin menjadi 'admin'.
  5. Akun demo siap digunakan untuk presentasi!
